# 物居 PWA — QR 码功能开发计划

## 概述
为物品和容器添加 QR 码生成（支持打印）、扫描识别、以及扫描关联功能。

## 数据格式
- 物品 QR: `wuju:item:<uuid>`
- 容器 QR: `wuju:container:<uuid>`

## Phase 1: QR 码生成
- [ ] 引入 qrcode 库（纯 JS，本地打包，无 CDN 依赖）
- [ ] 物品详情页添加「QR 码」按钮
- [ ] 容器详情页添加「QR 码」按钮
- [ ] QR 码弹窗（居中展示大图 + 物品/容器名称）
- [ ] 打印按钮（调用 window.print，隐藏无关元素）

## Phase 2: QR 码扫描（通用入口）
- [ ] 引入 html5-qrcode 或使用 BarcodeDetector API
- [ ] 底部 tab 添加「扫描」入口（📷）
- [ ] 扫描界面（相机取景框 + 关闭按钮）
- [ ] 解码后自动判断是物品还是容器，跳转到对应详情页
- [ ] 未知格式提示

## Phase 3: 扫描关联
- [ ] 物品详情页「关联物品」区域增加「📷 扫描关联」按钮
- [ ] 点扫描 → 打开扫描界面（仅匹配容器）
- [ ] 扫到容器 → 自动创建「属于」关联
- [ ] 关联成功提示（toast）

## Phase 4: 完善
- [ ] 扫描历史记录（可选）
- [ ] QR 码分享（Web Share API，可选）
- [ ] 批量打印（可选）
